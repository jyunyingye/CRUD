## API 說明

```
1. 取會員商品清單 
GET -- http://{host}/api/v1/Products

2. 取單筆商品 
GET -- http://{host}/api/v1/Products/{prdtuctid}

3. 新增商品資料 
POST -- http://{host}/api/v1/Products

4. 更新商品資料 
PUT -- http://{host}/api/v1/Products/{prdtuctid}

5. 刪除商品資料 
DELETE -- http://{host}/api/v1/Products/{prdtuctid}
```
